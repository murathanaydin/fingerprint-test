Server-Side Fingerprint Notu

Bu dosya, tarayıcı içinden doğrudan alınamayan fingerprint sinyallerini açıklar.

TLS fingerprint (JA3/JA4)
HTTP/2 SETTINGS frame
Header sırası ve HTTP değerleri
TCP/IP özellikleri (TTL, window size vb.)

Bu veriler JavaScript ile statik HTML sayfasından güvenilir biçimde alınamaz. Bunlar için sunucu tarafında proxy, reverse proxy, CDN logları veya özel bir backend gerekir.
